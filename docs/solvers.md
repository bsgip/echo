# Solvers

Echo uses [pyomo](https://pyomo.readthedocs.io/en/stable/index.html) to interface with third-party solvers. This means that Echo in principle supports all the solvers that pyomo [supports](https://pyomo.readthedocs.io/en/stable/getting_started/solvers.html).

Echo has been tested with [CPLEX](https://www.ibm.com/products/ilog-cplex-optimization-studio/cplex-optimizer), [Gurobi](https://www.gurobi.com/solutions/gurobi-optimizer/) and [CBC](https://github.com/coin-or/Cbc).
