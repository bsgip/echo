echo: Energy and Commodity Holistic Optimiser
==========================================================================
**echo** is a python-based multi-commodity energy system modelling tool
designed to answer grid integration questions.

This project is developed and maintained by the Battery Storage and Integration Program group at Australian National University.

.. note::
   This project is under active development.
